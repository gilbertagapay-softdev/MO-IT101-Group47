- **Role:** Developer

## Program Details
This program implements the MotorPH Payroll System required for MS2.  
It performs the following:

- Reads employee information  
- Matches attendance records  
- Computes payroll for one employee or all employees  
- Calculates gross pay  
- Applies deductions:
  - SSS  
  - PhilHealth  
  - Pag‑IBIG  
  - Withholding tax  
- Displays a complete breakdown of net salary  
- Runs through a console‑based Java menu
